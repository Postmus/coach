COACH Risk Engine
=================

COACH Risk Engine is a patient-level simulation model for predicting survival and hospitalization in patients with heart failure.

Requirements
------------

Java 6.

Versions
--------

1.0: Initial release.

Building from source
--------------------

To compile COACH Risk Engine from source, the following libraries are required:
- JFreeChart
